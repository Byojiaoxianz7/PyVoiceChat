# 使用方法
> 推荐使用 Anaconda

1, 首先把这个仓库 clone 下来 
```
$ git clone https://github.com/FatTig3R/PyVoiceChat.git
```
2, 安装一些必要的库
```
pip install -r requirements.txt
```

3, 运行
```
python3 main.py
```

不出意外的话就可以正常使用了
